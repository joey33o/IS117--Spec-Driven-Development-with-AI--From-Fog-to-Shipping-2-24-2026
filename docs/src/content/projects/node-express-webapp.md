## Overview

Built a full-stack web application that demonstrates the complete request-response cycle between a browser client and a Node.js server. The back-end is powered by Express and organized into a `server/` directory, while the front-end lives in `src/`.

## How It Works

The Express server handles incoming HTTP requests, processes them, and returns responses to the client. The project structure separates compiled output into a `bin/` folder, keeping source and build artifacts cleanly divided.

## Key Decisions

Using Node.js and Express on the same language stack (JavaScript) on both client and server reduces context-switching and accelerates development. This project established the foundation for understanding how real-world APIs communicate across the network layer.
