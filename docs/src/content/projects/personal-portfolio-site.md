## Overview

Built a personal website from scratch using only HTML and CSS — no frameworks, no build tools, no dependencies. The project lives in a `docs/` folder and is deployed directly from the repository.

## How It Works

Each page is a self-contained HTML file linked together through consistent navigation. CSS is loaded via a `<link>` tag with a validated file path, and the stylesheet defines all layout, typography, and color across the site.

## Key Decisions

Starting with zero dependencies forces a clear understanding of how the browser actually parses and renders markup. Debugging a broken CSS path — one of the most common early mistakes — reinforced the importance of correct relative path construction before reaching for any tooling.
